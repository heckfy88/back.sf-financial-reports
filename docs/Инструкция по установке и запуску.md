# Инструкция по установке и запуску

Для успешной работы приложения необходимо установить **docker** и **docker compose**.

Система состоит из двух частей: 

* **Backend**

Код приложения находится по следующей ссылке:  
https://github.com/heckfy88/back.sf-financial-reports/tree/develop

Необходимо выгрузить приложение в ветке **/develop** 
и запустить [docker-compose.yml](../docker-compose.yml) скрипт(включает загрузку postgres в docker).  
При необходимости, в скрипте поменять значения на необходимые.

[Описание настроек приложения](Описание%20настроек%20приложения.md)

Ветка содержит миграцию, заполняющую таблицы, для демонстрации работы приложения.  

После запуска приложения **swagger** доступен по ссылке:  
http://localhost:8080/swagger-ui/index.html  

* **Frontend**

Код приложения находится по следующей ссылке:  
https://github.com/heckfy88/front.sf-financial-reports/tree/master

Необходимо выгрузить приложение в ветке **/master** 
и запустить docker-compose.yaml скрипт. 
Запускать после сборки backend-части.  
При необходимости, в скрипте поменять значения на необходимые.  

Приложение после запуска доступно по ссылке(рекомендуется использовать браузер Chrome):  
http://localhost:3000
