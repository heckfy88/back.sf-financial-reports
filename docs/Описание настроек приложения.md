# Описание настроек приложения

Значения ссылаются на переменные, объявленные и определенные в файле [docker-compose.yml](../docker-compose.yml)

```yaml
spring:
  # Настройки подключения к базе данных
  datasource:
  # Путь подключения к базе данных
  url: ${SPRING_DATASOURCE_URL:jdbc:postgresql://localhost:5432/postgres}
  # Имя пользователя для поключения
  username: ${SPRING_DATASOURCE_USERNAME:ruslanagaev}
  password: ${SPRING_DATASOURCE_PASSWORD:1978}
  
# Настройки генерации и валидации JWT токена
jwt:
  # Секрет JWT токена
  secret: ${JWT_SECRET:thisisaverylongsecretkeywithenoughlength!!}
  # Система, выпускающая JWT токен
  issuer: ${JWT_ISSUER:financial-reports}
```
