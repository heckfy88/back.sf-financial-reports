# Финансовый мониторинг и отчетность

## Цель проекта

Разработка веб-приложения для мониторинга финансовых
показателей и генерации отчетов, которое позволит
пользователям отслеживать свои доходы и расходы, а также
получать аналитические отчеты на основе введенных
данных.

## Описание проекта

Проект - система, где пользователи могут вводить свои финансовые данные (доходы, расходы,
категории транзакций) и получать визуализацию своих финансовых показателей. 

Система предоставляет следующий функционал:  

* Авторизоваться в систему
* Просмотреть список транзакций авторизованного пользователя
* Добавить новую транзакцию
* Добавить категорию транзакции
* Редактировать существующую транзакцию
* Удалить существующую транзакцию
* Просмотреть отчеты по:
  1. Времени в разрезе  
  2. Типу транзакции
  3. Сравнению поступлений и трат
  4. Статусам
  5. Банкам отправителя и получателя
  6. Категориям доходов и расходов
* Получить данные с учетом фильтров
* Выгрузить список транзакций в файле
